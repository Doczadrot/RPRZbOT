# RPRZ Telegram Bot

![Python Version](https://img.shields.io/badge/python-3.9%2B-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)

Этот проект представляет собой Telegram-бота, разработанного для автоматизации и упрощения процесса учета несоответствий и брака на производстве РПРЗ. Бот не только регистрирует инциденты, но и предоставляет интеллектуальный поиск по нормативной документации (СТП) с использованием современных технологий обработки естественного языка (NLP) и Retrieval-Augmented Generation (RAG).

## 🚀 Основные возможности

- **Регистрация актов о браке**: Быстрое создание актов с возможностью прикрепления фотографий для визуального подтверждения.
- **Просмотр и поиск актов**: Удобный доступ к сохраненным актам с возможностью поиска по ним.
- **Интеллектуальный поиск по документации (RAG)**: Система позволяет задавать вопросы на естественном языке и получать точные ответы, основанные на содержании стандартов предприятия (СТП).
- **Векторная база данных**: Использование `FAISS` для быстрого и семантического поиска по текстовым данным.
- **Поддержка локальных LLM**: Интеграция с языковыми моделями, такими как Saiga и Vikhr, для обеспечения конфиденциальности и автономной работы.
- **Обработка PDF**: Автоматическое извлечение текста из PDF-документов для их последующей индексации и анализа.
- **Модульная архитектура**: Гибкая структура проекта, позволяющая легко добавлять новый функционал.
- **Тестирование и логирование**: Комплексная система тестов и подробное логирование для обеспечения надежности и простоты отладки.

## 🛠️ Технологии и библиотеки

- **Бэкенд**: Python
- **Telegram Bot API**: `pyTelegramBotAPI`
- **Нейронные сети и NLP**: `PyTorch`, `sentence-transformers`, `langchain`
- **Векторный поиск**: `FAISS`
- **Работа с данными**: `NumPy`, `PyPDF2`, `python-docx`
- **Окружение**: `python-dotenv`
- **Тестирование**: `unittest2`, `coverage`

## ⚙️ Установка и запуск

**1. Клонируйте репозиторий:**
```bash
git clone https://github.com/Doczadrot/RPRZbOT.git
cd RPRZ
```

**2. Создайте и активируйте виртуальное окружение:**
```bash
python -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate
```

**3. Установите зависимости:**
```bash
pip install -r requirements.txt
```

**4. Настройте переменные окружения:**

Создайте файл `.env` в корневой директории проекта и добавьте в него следующие переменные:

```
TELEGRAM_BOT_TOKEN="ваш_токен_для_telegram_бота"
GOOGLE_API_KEY="ваш_ключ_для_google_api"
```

**5. Запустите бота:**
```bash
python src/RPRZ.py
```

## ✅ Тестирование

Для запуска тестов и проверки покрытия кода выполните следующие команды:

```bash
# Запуск всех тестов
python -m unittest discover tests

# Запуск тестов с отчетом о покрытии
coverage run -m unittest discover tests
coverage report -m
```
