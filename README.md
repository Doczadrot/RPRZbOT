# RPRZ Telegram Bot

Этот бот предназначен для учета несоответствий на заводе РПРЗ.

## Функционал

### Основные возможности
*   Регистрация актов о браке с прикреплением фотографий
*   Просмотр сохраненных актов
*   Поиск по актам с использованием векторной базы данных

### RAG-система (Retrieval-Augmented Generation)
*   **Интеллектуальный поиск по документам СТП** - система может находить релевантную информацию в стандартах предприятия
*   **Векторная база данных FAISS** - быстрый семантический поиск по содержимому документов
*   **Нейропомощник с контекстом** - ИИ-ассистент, который отвечает на вопросы, основываясь на документах предприятия
*   **Поддержка PDF-документов** - автоматическая обработка и индексация технических стандартов
*   **Локальные языковые модели** - работа с моделями Saiga и Vikhr для обеспечения конфиденциальности

### Дополнительные функции
*   **Система тестирования** - полное покрытие тестами для надежности
*   **Логирование операций** - детальные логи для отладки и мониторинга
*   **Гибкая архитектура** - модульная система для легкого расширения функционала

## Запуск

1.  Установите зависимости: `pip install pyTelegramBotAPI`
2.  Замените `'YOUR_BOT_TOKEN'` в файле `RPRZBOT.py` на ваш реальный токен Telegram бота.
3.  Запустите бота: `python RPRZBOT.py`
